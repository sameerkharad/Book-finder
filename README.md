#  Book Finder

**Book Finder** is a modern React web application that helps users discover books by title, author, or subject.  
It integrates the **OpenLibrary API** to fetch book data dynamically and provides an interactive, responsive, and user-friendly interface.

---

##  Features

-  **Search Functionality** – Search books by title, author, or subject with a dynamic search bar and debouncing.  
-  **Favorites** – Add or remove books from favorites, stored in **Local Storage** for persistence.  
-  **Pagination** – Browse large results easily using Previous/Next navigation.  
-  **Scroll-to-Top** – Quickly jump back to the top of the page.  
-  **Responsive Design** – Optimized for mobile, tablet, and desktop devices.  
-  **Modern UI** – Styled with **Tailwind CSS** featuring a clean color scheme and elegant typography.  
-  **Loading & Error Handling** – Smooth loading animations and user-friendly error messages.  

---
##  Usage

- Enter a **title**, **author**, or **subject** in the search bar.  
- Click **Search** or wait for debounced results.  
- Use the  **Favorite button** to add or remove favorites.  
- Navigate results with **Previous/Next** buttons.  
- Scroll down and use the  **scroll-to-top button** for easy navigation.

---
##  Technologies Used

-  **React** – Frontend framework  
-  **Vite** – Fast build tool  
-  **Tailwind CSS** – Styling and layout  
-  **OpenLibrary API** – Book data source  
-  **Local Storage** – Favorites persistence
